## 三、云开发后台

### 6. 云数据库
在微信开发者工具「云开发 → 数据库」中创建/导入以下集合：
| 集合名 | 用途 |
|---|---|
| `campuses` | 嘉应学院各校区（江北、江南、梅州师范、医学院） |
| `sites` | 各校区内的地点（教学楼、宿舍、食堂等） |
| `categories` | 地点分类（教学、生活、运动等） |
| `schoolguides` | 校园指南文章 |
| `admins` | 管理员列表 |
| `media` | 轮播图、视频等媒体配置 |

### 7. 云存储
上传替换后的图片到云存储，并更新 `data/media.js` 和 `data/school.js` 中的图片链接。
