# 嘉应校园导航
moshunbin
嘉应学院校园导航微信小程序 —— 为师生和访客提供校园地图、地点查询、路线导航、校园指南等一站式校园服务。

## 项目简介

本项目是一个基于**微信小程序 + 云开发**的校园导航应用，覆盖嘉应学院江北校区、江南校区、梅州师范分教点、医学院四个校区。集成了腾讯地图 SDK 与和风天气 API，提供精准的校园位置服务和实时天气信息。

- **课题**：软件工程课题设计
- **成员**：黄浩城(231110271)、莫顺彬(231110293)、罗子钦(231110303)、黄湛文(231110306)、罗春妮(231110310)、朱祖勋(231110294)
- **指导老师**：魏彩珍

## 主要功能

### 🗺️ 地图导航
- 基于腾讯地图的校园交互式地图
- 实时定位与 POI 兴趣点展示
- 地点搜索、路线规划与导航
- 支持步行 / 驾车多模式路径规划

### 📍 地点汇总
- 按校区、分类浏览校园内所有地点
- 教学楼、宿舍楼、食堂、图书馆、运动场所、行政办公、校园景点、生活服务等分类
- 每个地点包含详细介绍、图片、位置信息

### 🏠 首页
- 轮播图展示校园风光
- 实时天气信息（和风天气 API）
- 常用功能快捷入口（地图导航、校园指南、地点汇总、地点热度排行等）
- 学校公众号 / 二维码快捷关注
- 地点热度排行榜

### 📖 校园指南
- 校园生活攻略文章
- 支持关键词搜索
- 图文并茂的内容展示

### ⚙️ 管理后台
- **地点管理**：校区 / 分类 / 地点的增删改查
- **媒体管理**：首页轮播图、视频、地点默认图片
- **信息管理**：校园指南文章编辑
- **成员管理**：管理员添加 / 删除（基于 openid 权限控制）

### 📝 更多
- 个人中心
- 软件声明与使用说明
- 用户反馈
- 管理员入口

## 项目启动

克隆这份代码后，根据你的情况选择对应的启动方式。

---

### 如果你是团队成员（已获得授权）

项目已配置好云环境和第三方服务，你只需：

#### 1. 确认授权

找项目负责人（黄浩城）做以下操作（一次性）：

- 把你添加为**小程序项目成员**（微信公众平台 → 成员管理）
- 把你添加为**云开发协作者**（云开发控制台 → 用户管理）
- 如需管理后台权限，让他把你的 openid 加入云数据库 `admins` 集合

#### 2. 安装微信开发者工具

下载安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

#### 3. 克隆并打开

```bash
git clone https://github.com/ggsmida1/JYU-Campus-Guide.git
cd 云开发项目
```

用微信开发者工具打开 `云开发项目` 目录，**无需修改任何配置**，直接点击编译即可运行。

#### 4. 运行

点击 **编译** → 模拟器运行；点击 **预览** → 手机扫码体验。

管理后台入口：小程序底部「更多」→「管理界面」。

---

### 如果你是独立开发者（没有共享云环境权限）

需要自行搭建全套环境：

1. 注册微信小程序账号，获取自己的 AppID
2. 开通微信云开发，获取云环境 ID
3. 修改 `app.js` 中的 `env` 为你的云环境 ID
4. 将所有云函数上传部署到你的云环境<br>（右键每个云函数目录 → 上传并部署：云端安装依赖）
5. 自行申请 [腾讯位置服务 Key](https://lbs.qq.com/) 和 [和风天气 Key](https://dev.qweather.com/)，修改 `data/map.js` 和 `data/data.js`
6. 通过管理后台初始化数据（校区、分类、地点等）

---

### Git 协作规范

1. 每人从 `main` 拉自己的功能分支开发，完成后提 PR 合并
2. 提交规范：`feat:` / `fix:` / `docs:` / `style:` / `refactor:`
3. 云函数修改后需手动上传部署到云环境<br>（右键云函数目录 → 上传并部署：云端安装依赖）
