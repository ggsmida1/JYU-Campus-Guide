# 嘉应校园导航

嘉应学院校园导航微信小程序 —— 为师生和访客提供校园地图、地点查询、路线导航、校园指南等一站式校园服务。

## 项目简介

本项目是一个基于**微信小程序 + 云开发**的校园导航应用，覆盖嘉应学院江北校区、江南校区、梅州师范分教点、医学院四个校区。集成了腾讯地图 SDK 与和风天气 API，提供精准的校园位置服务和实时天气信息。

## 主要功能

### 🗺️ 地图导航
- 基于腾讯地图的校园交互式地图
- 实时定位与 POI 兴趣点展示
- 地点搜索、路线规划与导航
- 支持步行 / 驾车多模式路径规划

### 📍 地点汇总
- 按校区、分类浏览校园内所有地点
- 教学楼、宿舍楼、食堂、图书馆、运动场所、行政办公、校园景点、生活服务等分类
- 每个地点包含详细介绍、图片、位置信息

### 🏠 首页
- 轮播图展示校园风光
- 实时天气信息（和风天气 API）
- 常用功能快捷入口（地图导航、校园指南、地点汇总、地点热度排行等）
- 学校公众号 / 二维码快捷关注
- 地点热度排行榜

### 📖 校园指南
- 校园生活攻略文章
- 支持关键词搜索
- 图文并茂的内容展示

### ⚙️ 管理后台
- **地点管理**：校区 / 分类 / 地点的增删改查
- **媒体管理**：首页轮播图、视频、地点默认图片
- **信息管理**：校园指南文章编辑
- **成员管理**：管理员添加 / 删除（基于 openid 权限控制）

### 📝 更多
- 个人中心
- 软件声明与使用说明
- 用户反馈
- 管理员入口

## 项目启动步骤

### 1. 环境准备

- 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 注册微信小程序账号，获取 AppID
- 开通微信云开发（在微信开发者工具中点击"云开发"按钮开通）

### 2. 克隆项目

```bash
git clone https://github.com/ggsmida1/-----.git
cd 云开发项目
```

### 3. 配置云开发环境

1. 用微信开发者工具打开项目目录
2. 在 `app.js` 中将 `env` 改为你自己的云开发环境 ID：
   ```js
   wx.cloud.init({
     env: '你的云开发环境ID',
   });
   ```
3. 在微信开发者工具中上传所有云函数：
   - 右键每个云函数目录 → **上传并部署：云端安装依赖**

### 4. 配置第三方服务

项目依赖以下第三方服务，需要自行申请 Key：

| 服务 | 配置文件 | 说明 |
|------|----------|------|
| 腾讯位置服务 | `data/map.js` → `mapKey` | [申请地址](https://lbs.qq.com/) |
| 和风天气 | `data/data.js` → `weatherKey` | [申请地址](https://dev.qweather.com/) |

### 5. 配置项目私有设置

在项目根目录创建 `project.private.config.json`（此文件已在 `.gitignore` 中）：

```json
{
  "projectname": "嘉应校园导航",
  "setting": {
    "compileHotReLoad": true
  }
}
```

### 6. 运行

1. 在微信开发者工具中点击 **编译** 按钮
2. 小程序将在模拟器中运行
3. 点击 **预览** 可在手机微信上扫码体验

### 7. 初始化数据

首次运行后，需要通过管理后台添加内容：
- 添加校区（江北校区、江南校区等）
- 添加地点分类（教学楼、食堂、宿舍等）
- 添加具体地点
- 配置轮播图、校园指南文章等

管理后台入口：小程序底部「更多」→「管理界面」（需先在云数据库 `admins` 集合中添加自己的 openid）
